GOLD-Python

The GOLD-Python proyect is a Python package with the objective of providing a simple and easy to use interface to the GOLD programming language’s finite automata library.

It supports Windows, Linux and Mac OS X, and works with Python 3.10 and 3.11.

def createAutomaton():
   Q = product(between(0, 7), between(0, 1))
   E = "01"
   Q0 = (0, 0)
   F = [(0, 0)]

   return DeterministicAutomata(Q, E, Q0, F, delta)

@deltafunc
def delta(x, y, next):
   d = int(next)
   if (x, y) == (0, 1):
      return (0, 1)
   elif x==7:
      return (0, 0) if (d-y==0) else (0, 1)
   else:
      return (x+1, (y+d) % 2)
